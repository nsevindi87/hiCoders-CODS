Document Objekt Model`in (DOM) Görüntülenmesi ve Degistirilmesi


A- DOM Agacinin görüntülenmesi

1- Node (dügüm) incelemesi

DOM agaci element paneli DOM ile iliskilendirilebiecek bütün aktivitelerin yer aldigi paneldir. Node ifadesi DOM icersindeki en temel nesnedir. DOM agacinda karsilasacak her detay ayri bir Node olarak kabul edilir. Döküman icersinde herhangi bir Node incelenmek istenirse panele ulasmanin farkli yollari mevcuttur.

Chrome DevTools Panelinin Acilma Sekilleri

  • Incelenmek istenilen Node secilip menü listesinden "Inspect" secenegi ile
  • Chrome Menu seceneklerinden More Tools + Developer Tools ile
  • CTRL + SHIFT + I kisayolu ile
  • F12 kisayolu ile

Panelin acilmasindan sonra bütün DOM yapisi ve Node`lar elements sekmesinde görüntülenebilir.
Özellikle bir kismin incelenmek istenilmesi halinde ikonu tiklanir.
Bu ikonun aktif hale gelmesinden fonra imlec ile incelenmek istenilen bolgeye tiklanarak o bölgeye ait Node larin getirilmesi saglanir.

2- DOM agacinda klavye ile gezebilme

DOM agacina girildikten sonra Node lar arainda klavye ile gezinebilmek mümkündür.
Yukari veya asagi oklar ile tek tek gezinilebilir.
Sol oka basildiginda secili olan Node`un kaynagi bir klasör gibi iceri taplanacaktir.
Sag oka basildiginda ise kapali olan bir Node icersinde yer alan diger nodelari görünür hale getirecektir.

3- Sayfa icerisinde Gezinme Durumunda

DOM agacinda inceleme yaparken sayfada baska bir bölümün acilmasi halinde secili Node kismina tekrar dönmek icin secili Node üzerine sag tiklanir ve "Scroll into view" sekmesi tiklanarak tekrar ilk incelenilen yere geri dönülebilir.

4- NODE arastirmasi

Elements sekmesinde bir string veya CSS gibi Node aramalari yapmak mumkundur. Klavyeden CTRL +F kisayolu kullanilarak arama gerceklestirilebilir.




B- NODE Düzenlemeleri

1- Icerik Düzenleme

Secili olan Node üzerine cift tiklama sonrasi icerik silinerek farkli bir icerik girilebilir. Bu degisiklik sayfada da dogrudan görüntülenecektir.

2-Özelliklerin Düzenlenmesi

Secili olan Node üzerinde Tag kismin cift tiklanarak secildikten sonra sag ok ile yan tarafina gelerek bu kisma yeni bir özellik eklenebilir.

3-Node Type Degisimi

Node icerisindeki Tag üzerine cift tiklanarak bu kisim baska bir özellik ile degistirilebilir halde gelir. Icerik degistirilince onaylanarak sayfaya yansimasi saglanir.

4- Node Siralamasi

Dom yapisinda bir node secildikten sonra sürükle birak yöntemi ile siralamasi degistirilebilir.

5-Güclendirme Durumu

Secili olan bir Node üzerine sag tiklayarak Force State > : hover sekmesi secilerek vurgulama saglanabilir.

6- Gizli Node

Secili olan Node üzerinde H tusuna basilarak Node gizlenmesi saglanir. Yine H tusuna basilarak gösterilmesi saglanir.

7-Node Silinmesi

Secilen Node delete tusuna basilarak silnebilir. Yine klavyeden CTRL+Z tuslari kullanilarak son yapilan silme islemi geri alinabilir.




C- Console Sekmesi Node Erisimi

1-Secili Olan Node'un $0 referanis

Secili olan Node uzerinden console

sekmesine gecilerek escapetusu ile $0 tanimlamasi yapilabilir.
*bunun pratikte ne ise yaradigini anladigim söylenemez.

2- Global Degisken Olarak Saklama

Bir Node birden fazla kullanilmak istenildiginde Node üzerine gelip sag tiklayarak Store as global variable

tiklanarak saklanabilir. Sonradan klavyeden temp1 yazilarak saklanan Node erisim saglanabilir.

3- JavaScript Kodu'nun Kopyalanmasi

Herhangi bir JS kodunun denenmek istenmesi kalinde Node üzerine sag tiklayarak Copy > Copy JS Path denilerek hafizaya alinir. Akabinde Console kismina gelinerek enter tusu ile yapistirilmasi mumkundur.



DOM Degisikliklerine Ara Verme

1- Özellik Degisiklerine Ara Verme

Node üzerinde sag tiklandiktan sonra menüden Break On > Attribute Modifications secenekleri tiklanir. Acilan pencerede durma dügmesine nastiktan sonra sayfanin hareket etmesi duracaktir. Tekrar hareket ettirebilmek icin ikonu tiklanmasi gerekmektedir.

2- Node Kaldirmaya Ara Verme

Bir Node silinmek ve ardindan duraklatilmak istenirse Node üzerine gelip sag tiklayarak Break On > Node Removal tiklanir. Akabinde DevTool sayfayi duraklatarak Node silinmesine neden olan kodu vurgular. devam edilebilmesi icin tiklanmasi gerekir.

3- Altsiralama Degisiklerine Ara Verme

Altsiralamaya bir araverme Node konulduktan sonra DevTool altsiralamaya herhangi bir sey eklendiginde veya kaldirildiginda sayfa duraklatilir. Bunun icin Break On > Subtree Modifications secilmesi gerekmektedir. Devam edilebilmesi icin ikonuna tiklanmasi gerekir.
